- **AI Interview Agent (Multi-Agent System | LiveKit + LangGraph)**  
  Designed and developed an adaptive behavioral interview agent using **multi-agent reasoning**,  
  **LLM fine-tuning**, and **metadata-driven session orchestration**.  
  Built transparent, bias-aware evaluation pipelines and deployed prototypes used in real interviews.

- **License Plate Recognition (ALPR) System ‚Äî YOLOv8 + LLM OCR**  
  Achieved **89.7% OCR accuracy** and **88.3% end-to-end recognition accuracy** on Saudi plates.  
  Implemented custom preprocessing (CLAHE, sharpening) and integrated with **AWS RDS**.

- **Arabic App Review NLP Classifier ‚Äî Hugging Face + BERT**  
  Built multi-class & multi-label classifiers for Arabic app reviews.  
  Performed EDA, preprocessing, label encoding, and fine-tuned a BERT-based model.  
  Evaluated using confusion matrices and classification reports.

- **Sign Language Recognition (Deep Learning | Encoder‚ÄìDecoder + Transformer)**  
  Designed a sequence-to-sequence model for sentence-level sign language translation.  
  Extracted features using **MobileNetV2**, implemented attention-based encoder-decoder models,  
  and built a **Transformer** variant for improved performance.

- **RAG Pipeline for the Ministry of Justice ‚Äî LlamaIndex + SageMaker**  
  Developed a production-ready RAG chatbot using LlamaIndex, LangChain/LangGraph, and Chroma.  
  Achieved **95% recall** with the e5 model and built a reusable Flask-based inference pipeline.

- **Machine Learning & Data Science Teaching Materials (2000+ Students)**  
  Created hands-on instructional content covering data analytics, preprocessing, visualization,
